# Advanced Calendar for Jira - XALT Design System

A user guide for the Advanced Calendar for Jira app, featuring the new XALT corporate design system.

## Design System

This project uses the **XALT Design System** which ensures consistent branding across all our web properties.

### Key Features
- Clean, professional design
- XALT brand colors and typography
- Responsive layout
- Smooth transitions and hover effects

### Colors
- **Primary:** `#081327` - Deep blue for backgrounds and headers
- **Accent:** `#01FFCD` - Bright cyan for CTAs and highlights
- **Secondary Accent:** `#2F9C86` - Muted teal for secondary elements
- **Background:** `#F9F9F9` - Light gray for page background
- **Text:** `#333333` - Dark gray for main text

### Typography
- **Headings:** Montserrat, sans-serif
- **Body:** Roboto, sans-serif

## File Structure
```
advanced_calendar_xalt/
```

## Quick Start

1. Open `index.html` in your browser
2. Navigate through the guide using the fixed navigation
3. Click cards to see hover effects
4. Responsive design works on mobile and desktop
